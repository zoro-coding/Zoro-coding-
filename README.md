# Zoro-coding-
🎯 Number Guessing Game  A simple Python console game where the computer picks a random number between 1–20 and the player has 5 attempts to guess it.  Hints: 📈 Too High / 📉 Too Low  Scoring: ✅ Player +1 if guessed correctly, ❌ Computer +1 if player fails  Replay option after every round  Fun, interactive, and easy to play 🎮
